# 参与方式（pre-1.0）

OSCA 处于 pre-1.0 阶段：**SPEC v0.3 已定稿**，v0.4 草案正由运行框架 Host 的参考实现互证喂养——
规范由真实使用演化，不由讨论演化。CLI（`lint / pack / load / replay`）与 Host 七组件均可用。

**现在欢迎：** 提 issue 讨论规范。建议按「场景 → 期望 → 对应规范章节」的结构描述。

**暂缓接受：** Pull Request。规范与参考实现仍在快速演化（蒸馏管道、界面、对接约定在路上），
等对接约定定稿后开放，届时补充贡献约定。

发现文档错误（错字、失效链接、示例不一致）例外——这类 PR 随时欢迎。
